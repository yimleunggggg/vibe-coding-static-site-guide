# YakuBus（yakushimabus.com）产品说明

> 对外介绍用 · 非技术版 · 2026-05

---

## 一句话

**YakuBus** 是面向屋久岛旅客的**独立公交查询工具**：在手机上一页查「下一班几点、怎么坐、多少钱」，并附官方路线图与上岛交通信息。日 / 中 / 英三语。

**网址**：https://yakushimabus.com

---

## 为什么做

屋久岛公交对第一次上岛的旅客并不友好：

- 时刻表密、站点多，纸质 PDF 在路边不好翻；
- 同一条路可能有不同运营公司，换乘规则不直观；
- 官方信息分散在多个 PDF 和网页里。

本站把**官方公开资料整理成可搜索、可筛选的查询页**，让旅客在港口、民宿或候车时，几十秒内得到「下一班」答案，而不是在包里翻 PDF。

---

## 核心功能（时刻表页）

面向**已在岛上、需要马上坐车**的场景。

| 功能 | 说明 | 出发点 |
|------|------|--------|
| **区间查询** | 选「从哪站 → 到哪站」，列出当日所有班次 | 旅客最关心的是「这一趟能不能到」，而不是整张表 |
| **下一班高亮** | 自动标出最近一班尚未发车的车 | 路边等车时的第一需求 |
| **按时间筛选** | 可选「只看当前时间之后」或指定时刻 | 避免在长列表里自己找 |
| **工作日 / 周六 / 日祝** | 按日本公交常见的日种切换 | 岛外旅客容易搞错「今天算哪种日」 |
| **常用区间快捷** | 宫之浦港↔安房等高频路线一键填入 | 多数旅客走固定几条线 |
| **站名搜索** | 48 个主要停留所可搜可点；站名以**日文为准**，中/英对照 | 站牌是日文，查车也应以日文站名为主 |
| **运营公司标识** | 班次旁标明种屋交通 / 松ばんだ等 | 不同公司票务规则不同，需一眼能辨 |

**数据依据**：屋久岛观光协会等发布的官方时刻表（含 2026 年 3 月改订版），页脚标注数据有效期与最近核对日期。

## 拓展功能一：路线与运价

**页面**：路线与运价（`/map/`）

| 功能 | 说明 | 出发点 |
|------|------|--------|
| **官方路线图** | 嵌入/展示屋久岛公交官方路线与运价 PDF（日 / 英） | 查「有哪些线、票价表长什么样」时仍需看图 |
| **手机看图模式** | 手机端以高清图片展示路线图，避免 PDF 打不开 | 岛上网络与手机环境参差，图片更稳 |
| **站点运价查询** | 选出发站、到达站，显示大人单程参考价 | 不必在整张运价矩阵里对坐标 |
| **乘车券与发售点** | 屋久岛「悠享满吃乘车券」价格及购买地点 | 多日乘车旅客常问，集中在一处 |
| **运价说明** | 整理券、找零、IC、儿童票等注意事项 | 官方 PDF 有，但旅客不会逐条读 |

**说明**：部分站点运价为根据官方锚点**估算**，页面会标注「目安」；实际以车内整理券与司机收费为准。

## 拓展功能二：上岛交通

**页面**：上岛（`/access/`）

| 功能 | 说明 | 出发点 |
|------|------|--------|
| **高速船 / 渡轮** | 鹿儿岛↔屋久岛等航班时刻与参考信息 | 公交只是岛上交通；上岛段同样容易查乱 |
| **机场巴士** | 与航班衔接的巴士信息 | 飞进飞出旅客的衔接需求 |
| **多日交通券** | 与 `/map/` 呼应的上岛侧说明 | 行程规划需要「进岛 + 岛内」一起看 |

本页以**表格直出**为主，不做复杂筛选，方便行前一次性浏览。

## 拓展功能三：信息与可靠度

| 功能 | 说明 | 出发点 |
|------|------|--------|
| **三语界面** | 日 / 中 / 英切换；站名仍以日文为主显示 | 国际旅客多，但现场站牌是日文 |
| **乘车要点** | 整理券、IC 限制、换乘、季节停运等 | 减少上车后才踩坑 |
| **官方信息链接** | 指向观光协会 PDF、运行状况页、荒川登山巴士官方 X 等 | 本站是整理层，权威仍以官方为准 |
| **荒川登山巴士动态** | 登山季（约 3–11 月）链至官方 X 速报 | 该线受天气影响大，静态表不够 |
| **关于本站** | 说明独立运营、更新方式、联系方式 | 建立信任，方便纠错 |

## 这是什么 / 不是什么

**是**

- 独立志愿者项目，数据来自**官方公开 PDF 与网页**的整理；
- 为**移动端优先**设计的查询页，可在 GitHub Pages 免费托管、打开即查；
- 持续随官方改表同步（通常每年 3 月、10 月前后有更新）。

**不是**

- 巴士公司或观光协会的**官方网站**；
- 实时 GPS 到站或购票系统；
- 法律意义上的「官方承诺时刻」——请以**车站站牌、车内广播与官方公告**为准。
