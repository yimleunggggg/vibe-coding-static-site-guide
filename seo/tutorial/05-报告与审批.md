# 怎么看报告 & 如何批准执行

## 日报（每天）

1. 打开 `docs/seo/reports/daily/最新日期.md` 或手机 ntfy
2. 先看 **§1 摘要** → **今日洞察** → **学一点**
3. **下一步建议**：🤖 已自动、👤 需你操作、⏳ 观察

新站 GSC 0 展示：看 GA4 是否有人用、索引是否 PASS，勿单日焦虑。

## 半月报（1/15）

- 文件 `*-reminder.md`，Issue 标签 `seo-round`
- 提示在 Cursor「跑一轮 SEO 优化」——适合要改 meta 时

## 周报（周一）

- `proposals/YYYY-WW-proposal.md` + Issue 标题 `seo-round YYYY-WW`
- 在 Issue 评论：
  - **`approve`** — 同意 §2 改动（执行 workflow 待接）
  - **`defer`** — 本周跳过
  - 或留言修改意见

**不必开 Cursor**；手机 GitHub App 即可。

| 行动 | 下一步（自动/手动标签） | 同左 |
